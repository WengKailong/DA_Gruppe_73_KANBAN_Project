Class: Task

Task(taskId, taskTitle, taskId, taskDiscription, taskDueDate, taskUrgency, taskAsignedTo, lastUpDate, lastUpDateBy)

This object class is designed to create and store all new task information needed for this KANBAN application.

Constructor

new Task(taskId, taskTitle, taskId, taskDiscription, taskDueDate, taskUrgency, taskAsignedTo, lastUpDate, lastUpDateBy)

Parameters:
Name Type Description
taskId Number this parameter is used to save the primary key as unique id of one task
taskTitle String this parameter is used to save title of a task
taskId String this parameter is used to save the information, to which category, this task belongs to
taskDiscription String this parameter is used to save the detailed discription of a task
taskDueDate String this parameter is used to save due date of one task
taskUrgency Number this parameter is used to save urgency of one task, 1 for Urgency 'high', 2 for urgency 'middle', 3 for urgency 'low'
taskAsignedTo Array this Array is used to save all user IDs, of which this task is asigned to.
lastUpDate String this parameter is used to store the time, when the task is last updated
lastUpDateBy Number this parameter is used to store the User ID, by who, the last update to task information was made.
